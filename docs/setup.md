# Development Setup

## Prerequisites
- Node.js 18+
- Azure CLI
- Git

## Azure Configuration
1. Resource Group: `rg-arboriq-dev`
2. Location: `australiasoutheast`

## Local Development
```bash
# Backend
cd backend
npm install
npm run dev

# Web
cd web
npm install
npm run dev

# Mobile
cd mobile
npm install
npm start
